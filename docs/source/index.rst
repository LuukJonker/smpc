Welcome to SMPCbox's documentation!
====================================

SMPCbox is a framework meant for the implementation of Secure Multi Party Computation (SMPC) protocols.
SMPCbox is foccused on providing insights and understanding to anyone interested in SMPC protocols.
SMPCbox provides the following key features:

* A framework for easy implementation of SMPC protocols.
* Distributed and simulated execution of implemented SMPC protocols.
* Visualisation of implemented SMPC protocols.
* Collection of statistics on protocol execution.
